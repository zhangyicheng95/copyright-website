# 版权保护官方网站

一个基于 HTML + CSS + JavaScript 开发的版权保护官方网站，提供版权信息查询、服务介绍和联系方式等功能。后端使用 Node.js + Express 提供 API 服务。

## 功能特点

- 响应式设计，适配各种屏幕尺寸
- 三个主要功能模块：版权信息、服务内容、联系我们
- 动态数据加载和展示
- 优雅的动画过渡效果
- 友好的用户界面

## 技术栈

### 前端
- HTML5
- CSS3 (使用 Grid 和 Flexbox 布局)
- JavaScript (原生 JS，使用 Fetch API)

### 后端
- Node.js
- Express
- CORS 跨域支持
